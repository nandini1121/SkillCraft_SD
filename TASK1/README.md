# 🌡️ Temperature Converter

A simple and interactive Temperature Converter web application built using HTML, CSS, and JavaScript.

## Features

✅ Convert temperatures between:
- Celsius
- Fahrenheit
- Kelvin

✅ User-friendly interface

✅ Responsive design

✅ Attractive modern UI with:
- Soft nude color palette
- Decorative doodles
- Gradient background
- Smooth animations

✅ Instant temperature conversion

## Technologies Used

- HTML5
- CSS3
- JavaScript
